# Task Management Board

## Overview

This is a simple Task Management Board built with React. It allows users to manage tasks, including creating, editing, deleting, and moving tasks between different statuses. Data is persisted in the browser's local storage to ensure it remains across page refreshes.

## Features

- Task creation, editing, and deletion
- Drag-and-drop functionality
- Task status management
- Local Storage persistence

## Installation

1. Clone the repository:
   bash
   git clone https://github.com/your-username/task-management-board.git
2. Navigate to the project folder:
   bash
   cd task-management-board
3. Install dependencies: (Node version: 22.11.0)
   bash
   npm install
4. Start the development server:
   bash
   npm start

## Technologies Used

- React
- Tailwind CSS
- React-DnD
- Local Storage
